# Toy Fraud Detector
This repository encompasses my self-directed dive into the basics of ML and some common Python ML frameworks.

### Overview
This is not meant to be a robust fraud detector, hence the name **toy** fraud detector. This is a purely educational venture.

I found a credit card fraud dataset on Kaggle which appeared to be a good candidate for a simple classification model. I chose to implement the same simple pipeline using three different frameworks so I could learn each of them at a surface level, and then I performed a simple comparative analysis on the three models to get a feel for key metrics.
